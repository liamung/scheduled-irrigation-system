# scheduled-irrigation-system
Irrigation system that controls flow to three distinct zones and a reservoir. Schedules specified zone irrigation on a 24 hour distribution with specifiable pump rpms through PuTTY. Programmed in C using an STM32 Nucleo-F401re.

Only the ioc file and the Core/Src/main.c file are necessary to port over the project. Must open PuTTY on the appropriate channel for the STM's cable.
